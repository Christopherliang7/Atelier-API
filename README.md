## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Christopherliang7/Systems-Design-Capstone.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start Server
   ```sh
   npm start
   ```
